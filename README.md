# Portfolio Website

This is a simple portfolio website project that showcases personal information, experience, skills, projects, and provides a contact form for users to get in touch.

## Project Structure

```
portfolio-website
├── index.html          # Main webpage for the portfolio
├── projects.html       # Projects overview page
├── contact_form.html   # Contact form for user inquiries
├── css
│   └── styles.css      # Styles for the portfolio website
├── js
│   └── main.js         # JavaScript for interactivity
├── assets
│   ├── fonts           # Directory for font files
│   ├── icons           # Directory for icon files
│   ├── profile-photo.jpg # Profile photo
│   ├── project1.jpg    # Image for project 1
│   ├── project2.jpg    # Image for project 2
│   etc.
├── projects
│   ├── project1.html   # Details for project 1
│   └── project2.html   # Details for project 2
└── demos
    ├── demo1.html      # Demo 1 page
    └── demo2.html      # Demo 2 page
```

## Features

- **About Me**: A section that provides a brief introduction.
- **Experience**: A section detailing professional experience.
- **Skills**: A section highlighting technical and personal skills.
- **Projects**: A section showcasing various projects.
- **Contact Form**: A form for users to send messages directly.

## Setup Instructions

1. Clone the repository to your local machine.
2. Open `index.html` in your web browser to view the portfolio.
3. Modify the content in `index.html`, `projects.html`, and `contact_form.html` as needed.
4. Customize styles in `css/styles.css` to fit your design preferences.
5. Add interactivity through `js/main.js`.
