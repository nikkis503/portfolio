## Features

### Home Page (index.html)
- **Banner Section**: Profile photo, name, tagline, and bio
- **Skills & Education**: 
  - Technical skills with progress bars (SQL, Python, Dashboards, AI Automation, Web Development)
  - Educational background
- **Featured Projects**: 4 key projects with external links
  - Auction Site for Trial & Error
  - Fans Voting Site for understan
  - Data Analytics Platform (Stanby)
  - AI Lab (AnyoneLab)
- **Keynotes Section**: 2 keynote presentations with downloadable PDF slides
- **Contact Section**: Email contact information

### Experience Page (experience.html)
- **Timeline Layout**: Visual timeline with job positions on the left and details on the right
- **3 Professional Roles**:
  - Senior Insight Analyst at The Z Label (02/2025 - Present)
  - Insight Analyst at Lane Crawford (01/2023 – 02/2025)
  - Social Media Analyst at Social Power Limited (11/2020 – 10/2022)
- **Company Descriptions**: Brief overview of each organization
- **Achievement Lists**: Bullet-pointed responsibilities and accomplishments

## Design & Styling

- **Color Scheme**: Navy blue (#000080) as primary color with white backgrounds
- **Typography**: Arial/Helvetica sans-serif font family
- **Layout**: 
  - Two-column layout on homepage (20% left sidebar, 80% right content)
  - Responsive design for mobile devices
- **Navigation**: Sticky top navigation bar
  - HOME | EXPERIENCE | CONTACT

## Technologies Used

- HTML5
- CSS3 (with custom styles, no frameworks)
- JavaScript (main.js)
- Navy blue pixel-art inspired design theme

## Setup Instructions

1. Clone or download the repository to your local machine
2. Open `index.html` in your web browser to view the portfolio
3. Navigate to different sections using the top navigation bar
4. All external project links open in new tabs

## Customization

1. **Profile Information**: Update name, tagline, and bio in `index.html`
2. **Skills**: Modify skill names and progress percentages in the Skills section
3. **Projects**: Update project images, descriptions, and links in the Featured Projects section
4. **Experience**: Add or modify job positions in `experience.html`
5. **Keynotes**: Add PDF files to the assets folder and link them in the Keynotes section
6. **Contact**: Update email address in the Contact section
7. **Styling**: Customize colors, fonts, and layout in `css/styles.css`
